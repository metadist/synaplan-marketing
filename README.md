# Synaplan Marketing

Landing pages and job postings for [Synaplan](https://github.com/metadist/synaplan) by metadist data management GmbH.

## Contents

- **`landing/`** — Landing page assets for synaplan.com
- **`jobs/`** — Job posting pages and social media assets (HTML pages, Open Graph images, social cards)

## Job Postings

Self-contained HTML pages with embedded styles, optimized for sharing on LinkedIn, Instagram, and other platforms. Each posting includes matching Open Graph and social media card images.
